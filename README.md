## ⚡ Профессиональное кэширование и экономия лимитов

В оригинальной версии Antigravity при отправке каждого нового запроса ИИ-модель заново перечитывает весь контекст вашего проекта (все открытые файлы, историю чата, структуру проекта и системный промпт). В нашем **Antigravity Client** реализована профессиональная технология **Prompt Caching** (кэширование контекста):
1. **Экономия до 3-5 раз:** При повторных запросах в рамках одного диалога закэшированная часть кода и файлов проекта считывается из сверхбыстрой серверной памяти практически бесплатно.
2. **Сравнение лимитов:** За счет умного кэширования наш базовый тариф **PRO** живет в **3–5 раз дольше**, чем стандартный официальный тариф Google AI Pro при одинаковом объеме написанного кода.
3. **Экономия времени:** Кэшированные ответы генерируются на **40-60% быстрее**, так как модели не нужно тратить время на повторный анализ структуры проекта.

## 🔌 Интеграция с Claude Code, Cursor и MCP (Model Context Protocol)

Для разработчиков, использующих CLI-ассистенты (например, **Claude Code**) или альтернативные редакторы (**Cursor**, **Windsurf**):

1. **Настройка прокси:** Наш клиент запускает локальный прокси на порту `8047`. Чтобы перенаправить трафик консольных утилит через шлюз, достаточно задать системные переменные окружения:
   * **Windows (PowerShell):**
     ```powershell
     $env:HTTP_PROXY="http://127.0.0.1:8047"
     $env:HTTPS_PROXY="http://127.0.0.1:8047"
     ```
   * **macOS / Linux (Bash/Zsh):**
     ```bash
     export HTTP_PROXY="http://127.0.0.1:8047"
     export HTTPS_PROXY="http://127.0.0.1:8047"
     ```
2. **Настройка в Cursor / VS Code:** В настройках редактора укажите альтернативный адрес API (Base URL): `http://127.0.0.1:8047/v1`.
